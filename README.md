
# Simple E-commerce Store

This repository contains the Simple E-commerce Store application source code.

## Open the project

To run the project locally:

1. Install dependencies:
   `npm install`
2. Start the frontend app:
   `npm run dev`
3. Open the browser at:
   `http://localhost:5173`

The backend API is located in the `backend/` folder and can be started separately if needed.

## About this repository

- Frontend: Vite + React
- Backend: Express + TypeScript
- Run the frontend from the project root with `npm run dev`
